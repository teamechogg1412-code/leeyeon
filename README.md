# FANSTAGE

b.stage 스타일 올인원 팬 플랫폼 MVP

- Home / From / Contents / Community / Shop
- 멤버십 권한 · 디지털 회원카드
- 데모 결제(주문) · 알림 · 오너 Admin

## Quick start

```bash
npm install
npx prisma migrate dev
npx tsx prisma/seed.ts
npm run dev
```

Open http://localhost:3000
